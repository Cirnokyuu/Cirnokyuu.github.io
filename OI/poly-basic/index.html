任意模数多项式乘法

如题，显然可以三模数 NTT 加上 CRT 来合并，但想想需要 999 次 FFT（6DFT+3IDFT），慢！

FFT 因为精度寄了，考虑拆系数 FFT，把 ai,bia_i,b_iai​,bi​ 拆成 215Ai+Bi,215Ci+Di2^{15}A_i+B_i,2^{15}C_i+D_i215Ai​+Bi​,215Ci​+Di​，得到四个系数均 ≤215\leq 2^{15}≤215 多项式 A,B,C,DA,B,C,DA,B,C,D，需要求 230A∗C+215(A∗D+B∗C)+B∗D2^{30}A * C + 2^{15}(A * D + B * C) + B * D230A∗C+215(A∗D+B∗C)+B∗D。

显然可以先 DFT 出来，做完点积 IDFT 回去，888 次 FFT，还是慢。

欲求 AAA 与 BBB 的点值，构造复多项式 P=A+iB,Q=A−iBP=A+iB,Q=A-iBP=A+iB,Q=A−iB，可以 DFT 做出 PPP 的点值后 O(n)O(n)O(n) 求 QQQ 的，具体为：

P[k]=A(wnk)+iB(wnk)=∑j(Aj+iBj)wnkj=∑jAj+iBj‾×wnkj‾‾=∑j(Aj−iBj)wn−kj‾=A(wn−k)−iB(wn−k)‾=Q[n−k]‾\begin{aligned} P[k]&=A(w_n^k)+iB(w_n^k)\\ &=\sum_j (A_j + iB_j)w_n^{kj}\\ &=\sum_j \overline{\overline{A_j + iB_j}\times \overline{w_n^{kj}}}\\ &=\sum_j \overline{(A_j - iB_j)w_n^{-kj}}\\ &=\overline{A(w_n^{-k})-iB(w_n^{-k})}\\ &=\overline{Q[n-k]} \end{aligned} P[k]​=A(wnk​)+iB(wnk​)=j∑​(Aj​+iBj​)wnkj​=j∑​Aj​+iBj​​×wnkj​​​=j∑​(Aj​−iBj​)wn−kj​​=A(wn−k​)−iB(wn−k​)​=Q[n−k]​​

知道了 PPP 和 QQQ 的点值，就可以解方程做出 AAA 与 BBB 的点值。整个过程只用 111 次 DFT。

接下来就是对 444 个多项式做 IDFT，仍然考虑把两个多项式 E,FE,FE,F 放在一起做 IDFT：构造复多项式 P=E+iFP=E+iFP=E+iF，其点值容易求出，而利用 EEE 和 FFF 都是实多项式的性质，所以对 PPP 进行 IDFT 之后得到的多项式，实部是 EEE，虚部是 FFF。整个过程只用 111 次 IDFT。

这样只用了 444 次 FFT（2DFT+2IDFT），比三模数 NTT 不知道高到哪里去了。

{f(x)∗g(x)≡1(modx⌈n/2⌉)f(x)∗f−1(x)≡1(modx⌈n/2⌉)\begin{cases} f(x) * g(x)&\equiv 1 \pmod{x^{\lceil n/2 \rceil}}\\ f(x) * f^{-1}(x)&\equiv 1 \pmod{x^{\lceil n/2 \rceil}} \end{cases} {f(x)∗g(x)f(x)∗f−1(x)​≡1(modx⌈n/2⌉)≡1(modx⌈n/2⌉)​

故 g(x)−f−1(x)≡0(modx⌈n/2⌉)g(x)-f^{-1}(x)\equiv 0 \pmod{x^{\lceil n/2 \rceil}}g(x)−f−1(x)≡0(modx⌈n/2⌉)，两边平方并乘上 f(x)f(x)f(x) 得到 f(x)∗g2(x)+f−2(x)∗f(x)−2f(x)∗g(x)∗f−1(x)≡0(modxn)f(x) * g^2(x) + f^{-2}(x) * f(x) - 2f(x) * g(x) * f^{-1}(x)\equiv 0 \pmod{x^n}f(x)∗g2(x)+f−2(x)∗f(x)−2f(x)∗g(x)∗f−1(x)≡0(modxn)。

化简得到 f−1(x)≡2g(x)−f(x)∗g2(x)(modxn)f^{-1}(x) \equiv 2g(x) - f(x) * g^2(x) \pmod{x^n}f−1(x)≡2g(x)−f(x)∗g2(x)(modxn)，此时 f−1(x)f^{-1}(x)f−1(x) 可以一次 NTT 求出。

※多项式牛顿迭代
