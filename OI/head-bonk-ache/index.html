P3447 [POI2006]KRY-Crystals

题意

给出序列 a1∼na_{1\sim n}a1∼n​，求满足以下条件的序列 bbb 的个数：

  • ∀i\forall i∀i，0≤bi≤ai0 \leq b_i \leq a_i0≤bi​≤ai​；
  • b1⊕b2⊕…⊕bn=0b_1\oplus b_2\oplus \ldots\oplus b_n=0b1​⊕b2​⊕…⊕bn​=0。

n≤50n \leq 50n≤50，ai<232−1a_i<2^{32}-1ai​<232−1，保证答案小于 2642^{64}264。

题解

如果前面 n−1n-1n−1 个数已经填好，那么第 nnn 个数只有一种选择，但它的上限会限制合法性。如果 an=+∞a_n = +\inftyan​=+∞，则方案数就是 ∏i=1n−1(ai+1)\prod_{i=1}^{n-1}(a_i+1)∏i=1n−1​(ai​+1)。

这启示我们从高位到低位考虑，不妨设所有 aia_iai​ 中最高的位为 www，aia_iai​ 包含最高位的 iii 的集合为 SSS。如果存在一个包含最高位的 aaa，它的 bbb 在 www 这一位不顶上界（即取 000）的情况，如果其非空子集 TTT 内的 bib_ibi​ 最高位填的是 000，也就是非最高位不受限，那么只需要从 TTT 集合中任取一个放最后「补漏」，其它随便取即可，方案数就是：

∑T⊆S,T≠∅[∣S∣−∣T∣ is even](2w)∣T∣−1∏i∉T(ai mod 2w+1)\sum_{T\subseteq S, T \neq \varnothing}[|S|-|T| \text{ is even}](2^w)^{|T|-1}\prod_{i \not\in T}(a_i\bmod 2^w+1) T⊆S,T​=∅∑​[∣S∣−∣T∣ is even](2w)∣T∣−1i​∈T∏​(ai​mod2w+1)

这个式子可以一遍 DP 简单地 O(n)O(n)O(n) 算出，不赘述。剩下的情况只有 SSS 内所有 bbb 最高位都填 111，判断合法性后就是将所有数都去掉最高位之后的子问题了。故我们做 O(log⁡m)O(\log m)O(logm) 次 DP 即可求出答案，时间复杂度 O(nlog⁡m)O(n \log m)O(nlogm)。

P7408 [JOI 2021 Final] ダンジョン 3

题意

n+1n+1n+1 个加油站，第 iii 与 i−1i-1i−1 个之间距离 aia_iai​ 个单位长度，第 iii 个加油站加单位油量需要花费 bib_ibi​。

车只能向右走，走 111 单位长度要用 111 单位油量，用完不能走。加油不能超过油量上限。

mmm 组询问，每次给出 l,r,xl,r,xl,r,x 表示一辆车从 lll 走到 rrr 并且油量上限为 xxx 需要花费的最小代价。

1≤n,m,ai,bi≤2×1051 \leq n,m,a_i,b_i \leq 2\times 10^51≤n,m,ai​,bi​≤2×105，1≤x≤1091 \leq x \leq 10^91≤x≤109。

题解

令油量上限为 uuu，有一个贪心：找到当前位置后第一个价格比它小的位置 xxx，加油至能走到 xxx 为止；若到 xxx 的距离 ≥u\ge u≥u，则加满 uuu 然后往后走一步。

考虑如何维护这个贪心：固定右端点，从右往左扫描左端点，维护价格递减的单调栈（为方便表述，令栈中元素是位置），栈顶即要找的位置。令当前位置为 iii，iii 加入单调栈后，策略的变化是用 bib_ibi​ 替换从 iii 到栈顶的前 uuu 个单位的油，不足则全部替换。原因是，我找到原单调栈中与 iii 距离 ≥u\ge u≥u 的第一个位置，到达此位置时我一定没油，后面策略就不变了。

但询问的 uuu 不等，就要讨论单调栈每一段的贡献。具体而言，假设我此时弹出 xxx，弹出后的栈顶为 yyy：

  • u≤x−iu\leq x-iu≤x−i：无贡献。
  • x−i<ux-i<ux−i<u：将 min⁡(u+i,y)−x\min(u+i,y)-xmin(u+i,y)−x 单位油的价格由 bxb_xbx​ 替换为 bib_ibi​，贡献 (bi−bx)(min⁡(u+i,y)−x)(b_i-b_x)(\min(u+i,y)-x)(bi​−bx​)(min(u+i,y)−x)。

容易发现这是个关于 uuu 的分段一次函数，线段树可以维护，这样就解决 Subtask 3 了。

对于区间 [l,r][l,r][l,r] 的询问，我们找出 [r−u,r][r-u,r][r−u,r] 中最小价格的位置 kkk，则 kkk 处的决策一定是加满油到走到 rrr 为止，这段路与「从 kkk 走到最后」的决策完全一致，所以 ans⁡(l,r)=ans⁡(l,+∞)−ans⁡(k,+∞)+bk(r−k)\operatorname{ans}(l,r)=\operatorname{ans}(l,+\infty)-\operatorname{ans}(k,+\infty)+b_k(r-k)ans(l,r)=ans(l,+∞)−ans(k,+∞)+bk​(r−k)。

只需要将右端点固定为最右边就可以算出 ans⁡(i,+∞)\operatorname{ans}(i,+\infty)ans(i,+∞)，时间复杂度 O(nlog⁡n)O(n \log n)O(nlogn)。
