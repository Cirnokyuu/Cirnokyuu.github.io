T1 树 (tree)

有一棵 nnn 个点的树，初始占领了两个节点 aaa, bbb。每一秒，一个被占领的节点可以选择一个相邻的未被占领的节点占领之。求占领整棵树至少需要多少秒。

n≤5×105n \leq 5\times 10^5n≤5×105。

查看提示

Tag\rm TagTag：DP，二分
Difficulty\rm DifficultyDifficulty：Easy

查看题解

如果只有一个关键点，那么显然可以设 dpidp_idpi​ 表示初始 iii 被染色，染完 iii 的子树的最小时间，O(nlog⁡n)O(n\log n)O(nlogn) 求出答案。也就是如果我们知道哪些点是 aaa 染的，哪些是 bbb 染的，求答案就简单了。

容易发现，只有 a∼ba\sim ba∼b 这个链上悬挂的一堆子树不知道是谁染的。观察到 aaa 与 bbb 染的越接近越优，则可以在链上二分找出最近的，复杂度 O(nlog⁡2n)O(n \log^2 n)O(nlog2n)。

瓶颈就在于排序了，而显然每次二分时只有 a∼ba\sim ba∼b 这个链上的 DP 值会改变，所以我们只需要最开始排序一次即可，复杂度优化至 O(nlog⁡n)O(n \log n)O(nlogn)。

T2 最小生成树 (mst)

给定一张 nnn 个点 mmm 条边的简单无向连通图，保证图的前 n−1n-1n−1 条边是图的一棵生成树。

你需要为每条边分配一个 1∼m1\sim m1∼m 的权值且每种权值只能用一次，求在所有前 n−1n-1n−1 条边是图的最小生成树的方案中，前 n−1n-1n−1 条边的权值和之和，对 109+710^9+7109+7 取模。

n≤20n \leq 20n≤20, m≤n(n−1)2m \leq \dfrac{n(n-1)}2m≤2n(n−1)​。

查看提示

Tag\rm TagTag：组合数学，状压 DP
Difficulty\rm DifficultyDifficulty：Medium

假设我们已经知道了前 n−1n-1n−1 条边的大小关系，那么非 MST 边带来的限制是什么？方案又如何求？

查看题解

假设我们已经用 O(n!)O(n!)O(n!) 的时间枚举了前 n−1n-1n−1 条边的大小关系，考虑答案如何求：这些边要作为 MST，那么一条其他的边 eee 不能环切之，也就是 eee 权值必须大于 MST 对应路径上边权的最大值。加上前 n−1n-1n−1 条边的大小限制，这 mmm 条边的限制就形如一条 n−1n-1n−1 的链上挂着一些儿子，最终拓扑序就是合法方案。

为方便叙述，转化成如下问题：有一个空队列，你会依次收到 w1w_1w1​ 个白球、一个黑球、w2w_2w2​ 个白球、一个黑球，依此类推。黑球只能插入至队首，白球可以插入至任意位置，最后黑球的位置之和即为答案。

假设我们 DFS 时记录了四元组状态 (n,b,c,s)(n,b,c,s)(n,b,c,s) 表示此时有 nnn 个球，其中有 bbb 个黑球，方案有 ccc 种，所有方案黑球位置之和为 sss，那么：

  • 添加一个白球，对于每种方案而言，都能衍生 n+1n+1n+1 种方案。考虑这些方案的黑球位置之和，第一部分是原先的 s(n+1)s(n+1)s(n+1)，第二部分是此白球贡献的 sss。故四元组转化为 (n+1,b,c(n+1),s(n+2))(n+1,b,c(n+1),s(n+2))(n+1,b,c(n+1),s(n+2))。
  • 添加一个黑球，则对于每种方案只能衍生出一种方案，并且 sss 增加 b+1b+1b+1。故四元组转化为 (n+1,b+1,c,s+c(b+1))(n+1,b+1,c,s+c(b+1))(n+1,b+1,c,s+c(b+1))。

而 DFS 时如何求出 www，我们按权值从大到小 DFS 选择该权值对应的边，加入这条边时所求即为：跨过这条边，但不跨过任一条已经选过的边的对应路径数。这个可以容斥后 O(m2nω)O\left(\dfrac {m2^n}\omega\right)O(ωm2n​) bitset 预处理、O(1)O(1)O(1) 求出。

将这个 DFS 改为状压 DP 即可，即记录 cS,sSc_S,s_ScS​,sS​ 表示已经从小到大选了 SSS 内的边时的 ccc 与 sss，不用记录 nnn 和 bbb 是因为知道了 SSS 之后二者很好求出。时间复杂度 O(m2nω+n2n)O\left(\dfrac{m2^n}{\omega}+n2^n\right)O(ωm2n​+n2n)。

T2 路径计数 (count)

444 个点的无向环，求从 111 出发到 111 结束，且经过第 iii 条边 viv_ivi​ 次的路径数。

v≤5×105v \leq 5\times 10^5v≤5×105。

查看提示

Tag\rm TagTag：欧拉回路。
Difficulty\rm DifficultyDifficulty：Medium

查看题解

一转无向图欧拉回路（两点间连 viv_ivi​ 条边），但无向图没法做。

我们直接对边定向，容易发现只需要钦定第一种边有多少正着的，有多少反着的，由于存在欧拉回路，其余的都容易递推出。然后直接套 BEST 定理完事。

T3 路径优化 (optimal)

nnn 行 mmm 列网格图，格点之间有无向带权边。给出一些格子为关键格，左上角必然是关键格，你需要求出包含所有关键格的环路的最短长度。

n,m≤400n,m \leq 400n,m≤400。

查看提示

Tag\rm TagTag：最短路，建模。
Difficulty\rm DifficultyDifficulty：Hard

不妨把左上角作为起点，那么最优环路和起点到每个关键点的最短路有什么关系？

查看题解

考虑从格点 (0,0)(0,0)(0,0) 出发，到任意一个关键格左上角的一条最短路径，它一定被包含在环路内。理由是，如果它露出了一部分在环路外，将环路扩展一定不劣。

那么我们建出最短路径树，这些边一定要被包含在环路内。最短路径树显然是个连通块，那限制就变成了路径不能穿过最短路径树。

不难想到对偶图，但似乎直接对偶似乎不太好做。我们把一个格点拆成左上、右上、左下、右下四个，之间连 000 的有向边；一个网格间的四个点，若相邻就连上对应边权。

  • 不能穿过 SPT：删除 SPT 穿过的边；
  • 包含所有关键格：删除关键格内的四个点。

(0,0)(0,0)(0,0) 的右上-左下最短路就是答案了。
