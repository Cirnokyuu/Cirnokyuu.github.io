割裂与分步转移

发表于 2022-07-18 分类于 OI Valine：

又到了喜闻乐见的社论环节，但这篇可能较为简单了。

抄袭自 dp 多维状态的优化 - chasedeath - 博客园。下面这段话讲的很好，我就搬过来了！

面对一个多维 DP 问题，根据维度之间联系的紧密程度，我们可以选择：

  1. 维度之间紧密相关，只能直接枚举；

  2. 维度之间完全无关，只是贡献通过某种形式相加，可以割裂为两个 DP 处理；

  3. 介于 111, 222 之间，不能割裂计算，但是可以将转移过程割裂为若干步来优化。

割裂

题意

给出 [1,n][1,n][1,n] 的所有子区间的权值，即 [i,j][i,j][i,j] 权值为 w(i,j)w(i,j)w(i,j)。称一个区间序列 [a1,b1],[a2,b2],…,[ak,bk][a_1,b_1],[a_2,b_2],\ldots,[a_k,b_k][a1​,b1​],[a2​,b2​],…,[ak​,bk​] 是合法的当且仅当对于 i∈[1,k)i\in[1,k)i∈[1,k) 都有 ai<ai+1a_i<a_{i+1}ai​<ai+1​ 且 bi>bi+1b_i>b_{i+1}bi​>bi+1​。此时定义其权值为 ∑i=1k−1w(ai−1,ai)+w(bi,bi−1)\sum\limits_{i=1}^{k-1}w(a_{i-1},a_{i})+w(b_{i},b_{i-1})i=1∑k−1​w(ai−1​,ai​)+w(bi​,bi−1​)，求合法区间序列的权值的最大值。

朴素 DP

dpa,bdp_{a,b}dpa,b​ 表示最后一个区间选择 [a,b][a,b][a,b] 时已有的最大权值，O(n4)O(n^4)O(n4)。

割裂

显然算贡献时无需同时知道 a,ba,ba,b 的配对关系，因为 aaa 的限制只与 aaa 的相邻项有关，bbb 只与 bbb 的相邻项有关。

我们将 aaa 与 bbb 的转移割裂开来，也就是分别选出 aaa 的递增序列、bbb 的递减序列，最后对位拼在一起。那么判断能拼在一起需要二者长度一致，还需要知道 aaa 的最后一项与 bbb 的最后一项的大小关系。

那么不妨 fi,jf_{i,j}fi,j​ 表示选了长度为 iii、结尾为 jjj 的 aaa 的递增序列的最大价值，gi,jg_{i,j}gi,j​ 同理。状态数 O(n2)O(n^2)O(n2)，转移 O(n)O(n)O(n)，最终合并答案需要 O(n2)O(n^2)O(n2)，总的复杂度是 O(n3)O(n^3)O(n3)。

分步转移

题意

给出 [1,n][1,n][1,n] 的所有子区间的权值，即 [i,j][i,j][i,j] 权值为 w(i,j)w(i,j)w(i,j)。称一个区间序列 [a1,b1],[a2,b2],…,[ak,bk][a_1,b_1],[a_2,b_2],\ldots,[a_k,b_k][a1​,b1​],[a2​,b2​],…,[ak​,bk​] 是合法的当且仅当对于 i∈[1,k)i\in[1,k)i∈[1,k) 都有 ai<ai+1a_i<a_{i+1}ai​<ai+1​ 且 bi>bi+1b_i>b_{i+1}bi​>bi+1​。此时定义其权值为 ∑i=1k−1w(ai−1,ai)+w(bi,bi−1)+∑i=1kw(ai,bi)\sum\limits_{i=1}^{k-1}w(a_{i-1},a_{i})+w(b_{i},b_{i-1})+\sum\limits_{i=1}^k w(a_{i},b_{i})i=1∑k−1​w(ai−1​,ai​)+w(bi​,bi−1​)+i=1∑k​w(ai​,bi​)，求合法区间序列的权值的最大值。

朴素 DP

dpa,bdp_{a,b}dpa,b​ 表示最后一个区间选择 [a,b][a,b][a,b] 时已有的最大权值，O(n4)O(n^4)O(n4)。

分步转移

由上个题加上了区间的权值，不太容易割裂了，但转移时两维的限制仍然没有必然联系。

技巧是先转移 aaa 这一维，再转移 bbb 这一维。具体地，另设 fi,jf_{i,j}fi,j​ 表示这个区间右端点为 jjj 时，如果我下一个区间左端点选 iii，已知贡献的最大值。

  • fff 由 dpdpdp 推导：枚举所有可能的下一个左端点，dpa,b+w(a,c)→fc,bdp_{a,b}+w(a,c)\to f_{c,b}dpa,b​+w(a,c)→fc,b​。
  • dpdpdp 由 fff 推导：枚举未选的右端点，容易发现贡献都可以被表示出，fa,b+w(a,c)+w(c,b)→dpa,cf_{a,b}+w(a,c)+w(c,b)\to dp_{a,c}fa,b​+w(a,c)+w(c,b)→dpa,c​。

这样转移就只用枚举左右端点中一个，复杂度 O(n3)O(n^3)O(n3)。

一道初赛题

定义函数 w(x)=popcount⁡(x)+xw(x)=\operatorname{popcount}(x)+xw(x)=popcount(x)+x，给一个长度为 nnn 的序列 aaa，选其一个子序列 bbb，使得 ∑w(bi⊕bi+1)\sum w(b_i \oplus b_{i+1})∑w(bi​⊕bi+1​) 最大。n≤105n \leq 10^5n≤105, ai≤216a_i \leq 2^{16}ai​≤216。

脑洞

将 aia_iai​ 看成其前 888 位与后 888 位组成的二元组。磷脂含说这个非常套路，并举出了一个例子，但我不这么觉得。唉！还是见识短浅了。

贡献看似可以割裂，但这俩必须绑起来选，就不那么好做了。那么一样的套路，fi,jf_{i,j}fi,j​ 表示我上一个后 888 位是 jjj，如果我此数前 888 位填 iii，已知贡献的最大值。转移同理，复杂度变成了 O(na)O(n\sqrt a)O(na​)。
